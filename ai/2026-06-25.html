OpenAI

GPT-5 Pro, 면역학자의 3년 묵은 난제 해결 — 공식 사례 공개

openai.com · 2026-06-24 공개 (연구는 2025년 말 수행)

OpenAI가 면역학자 Derya Unutmaz가 gpt-5 Pro로 항암 면역세포 관련 난제를 푼 과정을 공개했다. 모델은 deoxyglucose가 IL-2 단백질 합성을 방해한다고 제안했고, 미발표 데이터 기반 실험을 시뮬레이션해 CD8+ 세포의 림프종 살상력 증가를 정확히 예측했다.

"models like GPT-5 Pro function more like collaborators now"
제품 발표는 아니나, 프런티어 모델의 과학 연구 협업 활용을 보여주는 1차 사례.

Google (Gemini · DeepMind)

이미지 프리뷰 모델 오늘(6/25) 종료 — API 마이그레이션 필요

Google AI for Developers (Gemini API changelog) · 2026-06-25 종료

Gemini API에서 gemini-3.1-flash-image-preview와 gemini-3-pro-image-preview가 deprecated 처리되어 오늘부로 셧다운된다. 해당 프리뷰 모델을 호출하던 개발 워크플로는 정식(GA) 이미지 모델로 전환해야 한다. 동시에 gemini-3.5-flash GA, Managed Agents 퍼블릭 프리뷰 등 에이전트 기능이 이어지는 중.

Anthropic (Claude)

'Claude Tag' 베타 출시 — Slack 채널의 공동 작업자

anthropic.com · 2026-06-23

Slack 채널에서 @Claude를 호출해 작업을 위임하면, 단계별로 분해해 도구를 써가며 처리하고 스레드로 결과를 회신한다. 채널마다 하나의 Claude가 맥락을 누적 기억하며, 'ambient' 모드 시 능동적으로 후속 업데이트를 제공한다. claude-opus-4-8 구동, Enterprise·Team 대상 베타.

Anthropic, Alibaba의 Claude '무단 접근' 공개 고발

Bloomberg · 2026-06-24

Anthropic은 Alibaba가 4월 22일~6월 5일 약 25,000개 허위 계정으로 2,880만 건의 교환을 발생시키며 Claude의 핵심 역량(소프트웨어 엔지니어링·에이전트 추론)을 표적으로 대규모 무단 접근을 시도했다고 주장했다. 모델 역량 추출(distillation) 정황을 둘러싼 프런티어 랩 간 긴장을 보여주는 사례.
