AI 일일 브리핑

2026년 6월 11일 (목) · 최근 24시간 프런티어 AI 동향

TL;DR

  • OpenAI — gpt-5.5-instant가 ChatGPT 신규 기본 모델로 전환, 6/9 Go·Free 사용자에게 개인화 개선 배포.
  • Google — Gemini 3.5 시리즈의 gemini-3.5-flash 공개(에이전트·코딩 최강), Gemini API에 Managed Agents 퍼블릭 프리뷰.
  • Anthropic — Mythos급 신모델 Claude Fable 5 일반 출시(6/9 트래커 등재), Managed Agents 스케줄·CLI 퍼블릭 베타. 6/15 구독 프로그래밍 사용량 별도 크레딧 전환.
  • Meta·xAI·Mistral — 지난 24시간 내 1차 출처 기준 신규 발표 없음(직전 동향은 본문 주석).

OpenAI

GPT-5.5 Instant, ChatGPT 신규 기본 모델로 전환

OpenAI 공식 · 2026-06 (최근)

gpt-5.5-instant가 기존 gpt-5.3-instant를 대체해 전체 ChatGPT 사용자의 기본 모델로 배포됐다. 사실 정확성·간결성·이미지 이해·STEM 응답을 개선하고, 불필요한 되묻기·과도한 서식·이모지를 줄였다. "tighter, more direct answers without losing useful details."

개인화 개선, Go·Free 등급 확대 (6/9)

OpenAI Help Center · 2026-06-09

6월 9일 ChatGPT Go·Free 등급으로 개인화 개선이 배포됐다. 함께 공개된 차세대 메모리 시스템은 맥락 유지·선호 반영·최신성을 강화하며, 검토 가능한 메모리 요약 페이지가 미국 Plus·Pro에 순차 적용된다. 한편 gpt-4.5는 6/27, o3는 8/26 ChatGPT에서 은퇴 예정.

Google

Gemini 3.5 시리즈 시작 — 3.5 Flash 공개

blog.google · 2026-06 (최근)

구글이 Gemini 3.5 시리즈를 gemini-3.5-flash로 출발했다. 에이전트·코딩에 특화된 가장 강력한 모델로, 복잡한 장기 과제에서 gemini-3.1-pro를 핵심 벤치마크에서 앞선다고 밝혔다.

Gemini API, Managed Agents 퍼블릭 프리뷰

Google AI for Developers · 2026-06

Gemini API에 Managed Agents가 퍼블릭 프리뷰로 추가됐다. 구글 호스팅 격리 Linux 샌드박스에서 동작하는 자율·상태유지 에이전트를 개발·배포할 수 있다. 한편 gemini-3.1-flash-image-preview·gemini-3-pro-image-preview는 6/25 종료 예정.

Anthropic

Claude Fable 5 일반 출시 — Mythos급 신규 라인

Anthropic News · 2026-06-09 등재 [일부 미확인]

Opus 상위 티어로 자리매김한 Mythos급 모델 Claude Fable 5가 일반 출시됐다(LLM Stats 트래커 6/9 등재). 안전 분류기를 제외한 동급 모델 Claude Mythos 5는 Project Glasswing을 통해 제한 공급된다. 정식 발표 페이지 URL은 추가 확인 필요.

Managed Agents 스케줄·CLI 퍼블릭 베타 + 6/15 과금 변경

Claude Platform Release Notes · 2026-06

Claude Managed Agents가 cron 기반 스케줄 실행, vault 저장 환경변수, 브라우저 연동, 프라이빗 MCP 연결을 퍼블릭 베타로 지원한다. 6월 15일부터 구독 플랜의 프로그래밍 사용량은 별도 월간 크레딧 풀로 분리된다. Sonnet 4·Opus 4도 6/15 은퇴.
