OpenAI

GPT-5.5 Instant 응답 스타일·품질 개선

openai.com · 2026-06 · 공식 블로그

ChatGPT·API의 gpt-5.5-instant를 더 읽기 쉽고 자연스러운 대화체로 업데이트. "easier to read, more natural in everyday conversations, and better paced." 이번 변경으로 GPT-5.5 Instant·Thinking에서 캔버스는 제거되고, 글쓰기·코딩은 채팅 응답 내 writing/code 블록으로 직접 지원된다.

모델 정리: GPT-5.2 종료, GPT-4.5·o3 일몰 예정

help.openai.com · 2026-06-12 · ChatGPT 릴리스 노트

6/12부로 GPT-5.2(Instant/Thinking/Pro)는 ChatGPT에서 제거되고 기존 대화는 대응 GPT-5.5 모델로 자동 이전. GPT-4.5는 6/27, o3는 8/26 일몰 예정. 모델 라인업이 5.5 세대로 수렴 중.

API 컨테이너 세션 분당 과금 전환

developers.openai.com · 2026-06-02 · API 변경 로그

메모리 삭제·개인화 무료 확대

help.openai.com · 2026-06-09 · ChatGPT 릴리스 노트
