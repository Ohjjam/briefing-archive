AI 일일 브리핑

2026년 5월 18일 (월) · 최근 24~72시간 주요 동향

TL;DR

  • xAI — 일론 머스크가 Grok Build 에이전틱 코딩 CLI의 공개 베타 확장을 직접 푸시(5/14). Wall Street 주요 펌(Apollo, Morgan Stanley)이 Grok 내부 테스트 시작.
  • OpenAI — gpt-5.5-instant가 ChatGPT 기본 모델로 전환되며 환각률 52.5% 감소(고위험 프롬프트 기준). 미국 Pro 사용자 대상 개인 재무 계정 연동 기능 롤아웃 시작.
  • Anthropic — 법률 산업 전용 MCP 커넥터 20+종 및 12개 실무영역 플러그인 공개. PwC와 글로벌 단위 확대 파트너십 발표.
  • Google — DeepMind, AI 기반 새 입력 패러다임 Magic Pointer 발표(5/12, Chrome Beta 점진 출시). Chromebook을 'Googlebook'으로 리브랜딩 예정(2026년 가을).
  • Mistral — Mistral Medium 3.5 공개 + Vibe Remote Agents 출시. SWE-Bench Verified 77.6%. 유럽 은행 대상 사이버보안 도구 협의 중(5/13).

OpenAI

GPT-5.5 Instant, ChatGPT의 새 기본 모델로 전환

OpenAI 공식 블로그 · 2026-05-05

gpt-5.5-instant가 gpt-5.3-instant를 대체하며 전 사용자에 기본 모델로 적용. 내부 평가에서 의료·법률·금융 등 고위험 프롬프트의 허위 주장(hallucinated claims)이 52.5% 감소. Plus·Pro 사용자에는 웹·이미지·STEM 응답 품질 향상이 함께 적용됨.

ChatGPT 개인 재무 기능 단계적 롤아웃
