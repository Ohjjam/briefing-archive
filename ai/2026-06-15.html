OpenAI

ChatGPT 모델 라인업 개편 — gpt-5.2 제거, gpt-5.5로 이전

OpenAI Help Center · Model Release Notes · 2026-06-12

6월 12일자로 ChatGPT에서 gpt-5.2 계열(Instant·Thinking·Pro)이 제거되고, 기존 대화는 대응되는 gpt-5.5 모델로 자동 이전됐다. gpt-4.5는 30일 일몰 절차를 거쳐 6월 27일 ChatGPT에서 종료될 예정. 기본 모델이 gpt-5.5 중심으로 정리되는 흐름.

개인화 개선, ChatGPT Go·Free로 확대

OpenAI Help Center · ChatGPT Release Notes · 2026-06-09

6월 9일부터 ChatGPT Go·Free 사용자에게 개인화 개선이 순차 적용됐다. 기본 모델 응답이 더 정확해지고 답변이 간결·명료해져 사용자 맥락에 더 맞춰졌다는 설명. 유료 외 티어로의 품질 상향이 핵심.

Google (Gemini · DeepMind)

Gemini Enterprise — 6/16 일부 토글 종료, Asana 데이터스토어 프리뷰

Google Cloud · Gemini Enterprise Release Notes · 2026-06 (적용 6/16)

6월 16일부터 Global·US·EU 멀티리전에서 gemini-3.5-flash 기능 관리 토글이 사라진다(GA화 정리). 동시에 Asana 데이터스토어가 공개 프리뷰로 추가돼 자연어로 프로젝트·태스크를 검색·생성하는 워크플로 연동이 가능해졌다.

Gemini API — Managed Agents 공개 프리뷰, 이미지 프리뷰 모델 6/25 종료

Google AI for Developers · Gemini API Changelog · 2026-06

Google 호스팅 샌드박스에서 자율 실행되는 Managed Agents가 공개 프리뷰로 제공된다. 반면 gemini-3.1-flash-image-preview·gemini-3-pro-image-preview는 6월 25일 종료 예정이라 마이그레이션이 필요하다.

Anthropic (Claude)

오늘부터 Agent SDK·claude -p 별도 크레딧 과금 시작

Anthropic Help Center · Newsroom · 2026-06-15 (시행일)

6월 15일(오늘)부터 구독 플랜의 프로그래매틱 사용(Claude Agent SDK·claude -p)이 대화형 사용 한도와 분리된 별도 월간 크레딧에서 차감된다. API 표준 단가 기준 달러 표시 과금. 코드·에이전트 자동화를 많이 쓰는 사용자는 한도 체계 변화에 대비 필요.

美 수출통제 지령으로 Fable 5·Mythos 5 전 세계 차단

Fortune · Al Jazeera (1차 보도) · 2026-06-12~13

Anthropic · Claude Release Notes · 2026-06

Claude Code에 중첩 서브에이전트, 향상된 모델·리전 처리, 플러그인 검색, Chrome·VS Code·터미널 워크플로 개선이 반영됐다. 세션·권한·메모리 관련 다수 버그도 수정.

Grok Build 코딩 에이전트 베타 + 플러그인 마켓플레이스

xAI Docs · Release Notes / News · 2026-06 (이번 주)

Grok Imagine 1.5 Preview — 동영상 생성으로 확장
