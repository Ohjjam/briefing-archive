AI 일일 브리핑

2026년 6월 1일 (월) · 프론티어 AI 모델·기업 동향 · 최근 24~72시간

TL;DR

  • xAI가 Grok V9 Medium(1.5조 파라미터) 학습 완료를 알리며 6월 중순 공개를 예고 — 현재 SFT/RL 단계.
  • Anthropic의 6/15 과금 개편 임박: Agent SDK·claude -p·third-party 에이전트가 구독 한도에서 분리돼 별도 API 단가 크레딧으로 청구.
  • Google이 I/O 26에서 Gemini 3.5 Flash·Gemini Omni를 공개하고 Ultra 구독가를 $250→$200로 인하.
  • Mistral이 AI Now Summit에서 통합 에이전트 Vibe와 산업용 AI 스택, 자체 칩 설계 검토를 발표.
  • OpenAI는 GPT-5.5 Instant 품질 업데이트 진행, GPT-4.5 6/27 ChatGPT 퇴역 예정.

OpenAI

GPT-5.5 Instant 품질 업데이트 · GPT-4.5 ChatGPT 퇴역 예고

OpenAI Help Center (Model Release Notes) · 2026-05

gpt-5.5-instant가 ChatGPT·API에서 응답 스타일·가독성·페이싱 개선을 받았다. gpt-4.5는 30일 선셋 후 6/27 ChatGPT에서 퇴역, o3는 8/26 퇴역(API는 변경 없음). 구버전 정리와 신모델 강화가 동시에 진행 중이다.

GPT-5.4 mini 롤아웃 · 인터랙티브 시각 모듈 추가

OpenAI Help Center (ChatGPT Release Notes) · 2026-05

Free·Go 사용자도 + 메뉴의 "Thinking" 기능으로 gpt-5.4-mini를 쓸 수 있게 됐다. 특정 주제 질문 시 수식·변수를 실시간 조작하는 인터랙티브 시각 모듈도 제공된다.

Google (Gemini / DeepMind)

Gemini 3.5 Flash 공개 — 에이전트·코딩 최강 주장

blog.google · I/O 26 (2026-05-19)

Google이 gemini-3.5-flash를 출시. Terminal-Bench 2.1 76.2%, GDPval-AA 1656 Elo, MCP Atlas 83.6% 등에서 3.1 Pro를 앞선다고 밝혔다. 비디오 출력이 가능한 멀티모달 Gemini Omni와 24/7 개인 에이전트 Gemini Spark도 함께 공개.

Ultra 구독가 인하 · Developer 티어 신설

blog.google · 2026-05

Gemini Ultra 구독료를 월 $250 → $200로 인하하고, 엔지니어·전문가용 월 $100 Developer 티어를 신설했다. DeepMind는 Contextual AI 연구진 20여 명을 $80~90M 규모 라이선스 딜로 영입.

Anthropic (Claude)

6/15 구독 과금 개편 — 에이전트 사용량 별도 미터링

Anthropic News · 발표 2026-05-14, 시행 2026-06-15

Claude Agent SDK·claude -p·Claude Code GitHub Actions·third-party 에이전트가 구독 한도에서 분리돼 별도 달러 크레딧($20 Pro/$100 Max 5x/$200 Max 20x, 표준 API 단가, 롤오버 없음)으로 청구된다. 터미널 대화형 Claude Code, Claude.ai 웹/데스크톱/모바일, Cowork는 기존 구독 한도 유지.

Sonnet 4 / Opus 4 API 퇴역 · 신규 플랫폼 기능

Claude Platform Docs · 2026-06-15 예정

claude-sonnet-4·claude-opus-4가 6/15 API에서 퇴역하며 Sonnet 4.6/Opus 4.7로 마이그레이션 권고. Managed Agents 퍼블릭 베타, YAML 기반 ant CLI도 공개됐다.

Meta (Llama / Muse)

Muse Spark 공개 — Llama 후속 'Contemplating' 모드

9to5Mac · 2026-04-08 (원 발표 시점) [2차]

Meta가 AI 라인업을 리부트하며 멀티모달·추론·헬스·에이전트에서 경쟁력을 내세운 Muse Spark를 "스케일링 사다리의 첫 단계"로 발표. 신규 Contemplating 모드는 Humanity's Last Exam 58%, FrontierScience 38%를 기록했다고 주장. 6월 신규 발표는 확인되지 않음.

LlamaCon — Llama 보호 도구·Impact Grants

ai.meta.com (Meta AI Blog) · 2026

Llama Guard 4, LlamaFirewall, Llama Prompt Guard 2 등 오픈소스 보호 도구를 공개. 2회차 Llama Impact Grants로 10개 기관에 $1.5M+ 지원.

xAI (Grok)

Grok V9 Medium 학습 완료 — 6월 중순 공개 예고

xAI News / Elon Musk · 2026-05-25

1.5조 파라미터 Grok V9 Medium이 학습을 마쳤다(현 v8 small 0.5조의 3배). SFT 진행 중, RL 임박, 공개는 발표 후 2~3주(6월 중순) 목표. xAI는 매일 Grok 빌드 버전 업데이트를 공개 중.

Speech-to-Text API GA · Grok 4.1 전면 배포

xAI Docs (Release Notes) · 2026-05-30

25개 언어를 배치·스트리밍으로 전사하는 xAI Speech-to-Text API가 GA로 전환됐다. grok-4.1은 grok.com·X·iOS/Android 전 사용자에게 제공 중.

Mistral AI

통합 에이전트 'Vibe' 출시 — Work/Code 모드

mistral.ai · AI Now Summit (2026-05~06)

업무·코드를 아우르는 단일 에이전트 Vibe를 공개. Work Mode는 다단계 작업 계획을 사용자 승인 후 커넥터 전반에서 수행, Code Mode는 원격 코딩·PR 생성을 담당. VS Code 확장·CLI 업데이트 동반.

산업용 AI·데이터센터·자체 칩 설계 검토

VentureBeat · CNBC · 2026-05-28

물리 시뮬레이션을 결합한 'Mistral for Industrial Engineering'을 발표(Emmi AI 인수 기반, Airbus 협업). Les Ulis에 10MW 추론 전용 데이터센터(Q3 2026 가동)와 자체 칩 설계 검토도 공개. 엔터프라이즈용 Forge 플랫폼도 소개.

기타 주목할 만한 동향

프론티어 랩 전반 — '빠른 릴리스' 경쟁 가속

업계 종합 · 2026-05~06
