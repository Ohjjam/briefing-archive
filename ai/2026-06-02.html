AI 일일 브리핑

2026년 6월 2일 (화) · 최근 24~72시간 프런티어 AI 동향

TL;DR

  • Anthropic, 6월 15일부터 구독제 프로그래매틱(Agent SDK·claude -p·Claude Code GitHub Actions) 사용량을 별도 크레딧 풀로 분리 — 대화형 Claude.ai·터미널·Cowork는 영향 없음.
  • Google, gemini-3.5-flash 정식 출시 + AI Ultra 구독료 $250→$200 인하. gemini-3.5-pro는 6월 중 공개 예정.
  • OpenAI, gpt-5.4-mini를 ChatGPT 무료·Go 사용자에게 롤아웃. gpt-4.5는 6월 27일 ChatGPT에서 은퇴.
  • xAI, Speech-to-Text API 정식 출시 + Grok Web 커넥터 추가. Grok V9 Medium(1.5T 파라미터) 공개 임박.
  • Mistral, AI Now Summit에서 통합 에이전트 Vibe·산업용 AI 스택·자체 칩 설계 검토 발표.

OpenAI

GPT-5.4 mini, ChatGPT 무료·Go 사용자에 롤아웃

OpenAI Help Center · 2026-05 말~06-02

gpt-5.4-mini가 무료·Go 사용자의 "Thinking" 기능으로 제공된다. 유료(Plus/Pro)에서는 gpt-5.4 Thinking의 레이트리밋 도달 시 폴백으로 동작해 고사용 구간에서도 추론 접근성을 유지한다.

GPT-5.3 Instant 톤 개선 · GPT-4.5 6월 27일 은퇴 예고

OpenAI Help Center · 2026-05~06

gpt-5.3-instant 업데이트로 후속 응답 톤이 개선되고 티저성 표현이 줄었다. 한편 gpt-4.5는 30일 선셋 후 6월 27일 ChatGPT에서 은퇴하며, o3도 8월 26일 은퇴 예정이다.

Google

Gemini 3.5 패밀리 — Flash 출시, Pro 6월 중 공개

blog.google · 2026-05~06

Google이 "frontier intelligence with action"을 내건 gemini-3.5 패밀리를 공개했다. gemini-3.5-flash가 먼저 출시돼 에이전트·코딩에서 프런티어 성능을 내고, 무거운 gemini-3.5-pro는 6월 중 정식 출시 예정이다.

AI Ultra 구독료 $250→$200 인하 · Gemini 앱 9억 MAU

Google Cloud Blog · 2026-05~06

I/O 26과 함께 Ultra 구독료를 월 $200로 내렸다. Gemini 앱은 월 9억 MAU를 돌파(전년 4억의 2배 이상), Search의 AI Mode는 별도로 월 10억 MAU를 넘겼다.

Anthropic

6월 15일 구독제 빌링 개편 — 에이전트 사용량 별도 크레딧 풀

Anthropic News · 2026-06-15 시행 예고

6월 15일부터 Claude Agent SDK·claude -p·Claude Code GitHub Actions·서드파티 에이전트가 구독 한도에서 분리돼 별도 월 크레딧($20 Pro / $100 Max 5x / $200 Max 20x, API 정가 과금·이월 없음)으로 측정된다. 대화형 Claude.ai·터미널 Claude Code·Cowork는 영향 없음.

Claude Sonnet 4 · Opus 4 API 6월 15일 은퇴

Claude Platform Docs · 2026-06-15 예정

Anthropic이 claude-sonnet-4·claude-opus-4의 API 은퇴를 6월 15일로 예고하고, 각각 claude-sonnet-4-6·claude-opus-4-7로의 마이그레이션을 권고했다. 관리형 에이전트 하니스(Claude Managed Agents)도 퍼블릭 베타로 출시됐다.

Meta

오픈소스 Llama 전환 — Muse 시리즈로 이동[2차]

9to5Mac · 원 발표 2026-04, 06-02 시점 신규 업데이트 없음

지난 24시간 기준 Meta 측 신규 발표는 확인되지 않았다. 배경: 1분기에 Llama 오픈소스 전략을 종료하고 Meta Superintelligence Labs의 Muse Spark(코드명 Avocado, Contemplating 모드)를 첫 작품으로 내놨다. 1차 출처 미확인으로 2차 표기.

xAI

Speech-to-Text API 정식 출시 · Grok Web 커넥터 추가

xAI News · 2026-05 말~06-02

25개 언어 배치·스트리밍 전사를 지원하는 STT API가 GA됐다. Grok Web에는 SharePoint·Outlook·OneDrive·Google Workspace·Notion·GitHub·Linear 커넥터가 추가됐고, Imagine API에는 고품질 모드가 들어갔다.

Grok V9 Medium(1.5T) 공개 임박 — 일일 빌드 공개 중

xAI · 머스크 발언 2026-05-25

머스크가 5월 25일 Grok V9 Medium(1.5조 파라미터, v8 small의 3배) 학습 완료를 밝혔고, 공개 시점을 2~3주 뒤(6월 중순 전후)로 제시했다. xAI는 출시가 가까워지며 일일 Grok 빌드 버전을 공개하고 있다.

Mistral AI

AI Now Summit — 통합 에이전트 'Vibe' 출시

Mistral AI News · 2026-05 말~06-02

Mistral이 업무·코드 통합 에이전트 Vibe를 공개했다. 장기 작업용 Work Mode, 원격 코딩·PR용 Code Mode, VS Code 확장·CLI를 묶어 프로젝트 전반 자동화를 노린다.

산업용 AI 확장 · 자체 칩 설계 검토

CNBC / Mistral · 2026-05-28

Mistral for Industrial Engineering(Emmi AI 인수 기반 물리 시뮬레이션 결합)을 발표하고 Airbus와 협업을 공개했다. CEO는 인프라 강화 차원에서 자체 칩 설계를 검토 중이라고 밝혔으며, Les Ulis 10MW 추론 데이터센터를 Q3 가동 예정이다.
