AI 일일 브리핑

2026-07-13 (월) · 프론티어 AI 모델·기업 최신 동향

TL;DR

  • OpenAI가 gpt-5.6 패밀리(Sol/Terra/Luna)와 업무 에이전트 "ChatGPT Work"를 출시했고, MS 365 Copilot의 기본 모델로도 채택됐다.
  • Google DeepMind는 아키텍처를 처음부터 재설계한 gemini-3.5-pro 출시를 7월 17일로 연기했다고 밝혔다.
  • Anthropic은 Claude Enterprise 분석 강화, M365 커넥터 쓰기 권한, Cowork 웹/모바일 확장, Claude for Government 베타를 공개했고 claude-fable-5 무료 접근 기한을 7/12로 연장했다.
  • Meta는 에이전틱 코딩 모델 muse-spark-1.1과 이미지 생성 Muse Image를 공개하고, 자체 클라우드 사업 "Meta Compute"에 진출했다.
  • xAI는 코딩·에이전트 특화 grok-4.5를, Mistral은 신규 오픈웨이트 모델의 얼리 액세스와 로보틱스 모델 Robostral Navigate를 발표했다.

OpenAI

GPT-5.6 패밀리 공개, ChatGPT Work 에이전트 동시 출시

openai.com · 2026-07-09

OpenAI가 gpt-5.6 모델 패밀리(Sol·Terra·Luna)를 ChatGPT, Codex, API 전반에 순차 배포했다. Sol은 최상위 지능, Terra는 비용 대비 균형, Luna는 대규모·저비용 워크로드용이다. 동시에 연결된 앱과 파일의 맥락을 모아 문서·스프레드시트·프레젠테이션을 만드는 에이전트 "ChatGPT Work"도 출시했다.

"Sol is 54% more token efficient on agentic coding tasks" — Sam Altman (CNBC 인터뷰)

GPT-Live 음성 모델 및 MS 365 Copilot 통합

help.openai.com (공식 릴리스 노트) · 2026-07-08~09

실시간 음성 대화를 자연스럽게 만든 신규 풀듀플렉스 음성 모델 GPT-Live-1 및 mini가 유·무료 사용자 기본 음성으로 적용됐다. 또한 gpt-5.6이 Microsoft 365 Copilot(Word·Excel·PowerPoint·Chat·Cowork)의 새 기본 모델로 채택됐다.

Google

Gemini 3.5 Pro 출시 7월 17일로 연기 — 아키텍처 전면 재설계

deepmind.google · 2026-07-11 (보도: BigGo Finance) [2차]

Google DeepMind가 기존 gemini-2.5-pro 아키텍처를 버리고 처음부터 다시 설계한 gemini-3.5-pro의 출시를 7월 17일로 연기했다. 200만 토큰 컨텍스트 윈도, "Deep Think" 추론 레이어, 자율 워크플로 기능이 핵심이며 수학적 추론과 이미지 품질 개선을 목표로 GPT-5.6·Fable 5와 경쟁할 계획이다.

Nano Banana 2 Lite, Gemini Omni Flash API 프리뷰 공개

ai.google.dev (Gemini API 릴리스 노트) · 2026-07 초

가장 빠르고 비용 효율적인 이미지 생성 모델 "Nano Banana 2 Lite"가 출시됐고, 네이티브 멀티모달 모델 "Gemini Omni Flash"가 동적 비디오 워크플로 구축을 위해 API 퍼블릭 프리뷰로 제공된다.

Anthropic

Claude Enterprise 분석 강화 및 M365 커넥터 쓰기 권한 추가

anthropic.com/news · 2026-07

Claude Enterprise에 그룹·사용자별 사용량과 비용을 함께 보여주는 관리자 분석 대시보드, 모델별 권한 설정, 지출 알림 기능이 추가됐다. Microsoft 365 커넥터는 검색을 넘어 이메일 작성·발송, 일정 관리, OneDrive/SharePoint 파일 생성·수정까지 가능한 쓰기 도구가 활성화됐다.

Cowork 웹·모바일 확장, Claude for Government 베타 출시

support.claude.com (공식 릴리스 노트) · 2026-07

Claude Cowork가 웹과 모바일로 확장되어 기기 간 세션·파일을 공유한다. Claude for Government 베타가 오늘부터 제공되며, Anthropic이 계약·과금 주체를 맡아 기관이 별도 클라우드 계약 없이 이용할 수 있다. 설정 메뉴에는 월간 사용 리캡을 보여주는 "Reflect" 대시보드도 신설됐다.

Fable 5 무료 포함 접근 기한 7월 12일로 연장

anthropic.com · 2026-07-12 마감

유료 Claude 플랜에서 추가 비용 없이 제공되던 claude-fable-5 접근 기한이 당초 7월 7일에서 7월 12일 밤 11시59분59초(태평양시)로 연장됐다.

Meta

Muse Spark 1.1 공개 — 에이전틱 코딩 강화

ai.meta.com/blog · 2026-07-09

Meta Superintelligence Labs가 개인 에이전틱 작업과 다중 앱 오케스트레이션에 최적화된 muse-spark-1.1을 퍼블릭 프리뷰로 공개했다. 100만 토큰 컨텍스트를 능동적으로 관리하며, 대규모 코드베이스의 버그 진단·기능 구현·마이그레이션 성능이 크게 향상됐다. 저커버그는 OpenAI·Anthropic 대비 "공격적인 가격 정책"을 예고했다.

이미지 생성 모델 Muse Image 공개, Meta Compute 클라우드 사업 진출

Bloomberg · 2026-07-07

Meta AI 챗봇과 Instagram·WhatsApp에 텍스트 기반 이미지 생성·편집 모델 "Muse Image"가 탑재됐다. 또한 Nvidia와 복수 연도·기가와트 규모 칩 공급 계약을 확대하며 잉여 AI 인프라를 판매하는 신규 클라우드 사업 "Meta Compute"를 발표해 AWS·Azure·Google Cloud와 직접 경쟁하게 됐다.

xAI

Grok 4.5 출시 — 코딩·에이전트 특화 모델

x.ai/news · 2026-07-08

xAI가 코딩·앱 빌딩, 사무·서기 업무, 리서치, 글쓰기 등 지식노동 자동화를 겨냥한 grok-4.5를 출시했다. 입력 100만 토큰당 $2, 출력 $6로 가격을 책정했으며, 경쟁 모델 대비 "토큰 효율이 2배"라고 주장했다. 일론 머스크는 "Opus급 모델"이라 평가했다.

Grok Voice 신규 음성 21종 추가, Grok Build 세션 UX 개선

docs.x.ai (공식 릴리스 노트) · 2026-07

기존 5종에 이어 다국어 지원 플래그십 음성 21종이 실시간 Voice Agent API, TTS API, 신규 Grok Voice Agent Builder에 추가됐다. Grok Build는 스크롤·검색 개선과 기본 툴콜 그룹화, 빨라진 Plugins 탭 업데이트 흐름을 도입했다.

Mistral AI

신규 오픈웨이트 모델 얼리 액세스 시작, Robostral Navigate 공개

mistral.ai/news · 2026-07

Arthur Mensch CEO가 확인한 "fat but sparse" MoE 구조의 신규 오픈웨이트 모델이 연구·정부·산업 파트너 대상 얼리 액세스에 들어갔다. 별도로 단일 카메라와 자연어 프롬프트만으로 로봇이 복잡한 환경을 탐색하게 하는 하드웨어 비종속 로보틱스 모델 "Robostral Navigate"도 발표했다.

mistral.ai/news · 2026-07-03

Apache-2.0 라이선스의 오픈 모델 Leanstral 1.5가 공개돼 PutnamBench 672문제 중 587문제를 해결하며 형식 검증 벤치마크에서 강세를 보였다. Hugging Face와 무료 API로 제공된다.

"최고 모델"에서 "최적 적합"으로 — 업계 경쟁 구도 변화

llm-stats.com (업계 동향 트래커) · 2026-07-13

7월 9일 OpenAI·xAI·Meta가 같은 날 신모델을 발표하며 경쟁이 격화됐다. 업계 논조는 순수 벤치마크 점수 경쟁에서 가격·속도·접근성·실사용 적합도를 중시하는 "최적 적합" 경쟁으로 이동하고 있다는 평가다. 개발자 생태계에서는 PyTorch 2.13의 Apple Silicon 스파스 어텐션 가속(약 12배)도 주목받았다.
