grok-4.5, 코딩·에이전트 작업 중심으로 출시

SpaceXAI · 2026-07-16

SpaceXAI는 grok-4.5를 Grok Build의 기본 모델로 지정하고 API와 Cursor에도 제공하기 시작했다. 회사 발표 기준 출력 속도는 80 TPS, API 가격은 입력 100만 토큰당 2달러·출력 6달러다.

“Grok 4.5 is available today in Grok Build, in Cursor on all plans, and from the SpaceXAI console.”

Grok Automations: 일정·이메일로 자율 작업 실행

SpaceXAI · 2026-07-16

사용자가 작업을 한 번 설명하면 지정 일정이나 수신 이메일 조건에 따라 Grok이 실행하고 결과를 대화 기록·이메일·앱 알림으로 돌려준다. 예약 실행은 전체 사용자, 이메일 트리거는 SuperGrok에 제공된다.

“Describe the work once, choose when it runs, and Grok takes it from there.”
