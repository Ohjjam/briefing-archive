GPT-5.6 Sol·Terra·Luna 제한 프리뷰 — 정식 출시 "수주 내" 예고

OpenAI 공식 · 2026-06-26 발표 (진행 중 스토리)

OpenAI가 플래그십 GPT-5.6 Sol, 저비용 Terra, 최저가·최고속 Luna로 구성된 차세대 모델군을 공개했다. 현재는 API·Codex를 통해 소수 신뢰 파트너·기관(약 20곳)에만 제공되며, 미국 정부와 릴리스 계획을 공유한 뒤 "generally available in the coming weeks"로 정식 출시를 예고했다. 소프트웨어 엔지니어링·컴퓨터 사용·과학 연구·사이버보안 영역의 프런티어를 겨냥한다. 5일 전 발표지만 정식 출시가 임박한 개발 진행형 이슈로 추적한다.

Google

Gemini API — 구형 Veo·이미지 모델 지원 종료(6/30) 발효

Google AI for Developers (Gemini API 릴리스 노트) · 2026-06-30

Gemini API 릴리스 노트에 따르면 일부 구형 영상 생성 모델이 6월 30일자로 셧다운되어 Veo 3.1 프리뷰 모델 ID로 이전하도록 안내됐다. 앞서 gemini-3.1-flash-image-preview·gemini-3-pro-image-preview도 6/25 종료됐다. 한편 GA 버전 Gemini 3.5 Flash가 이제 gemini-flash-latest가 가리키는 기본 모델이다. 신규 모델 출시라기보다 플랫폼 정비 성격의 변경이다.

Anthropic

Claude Sonnet 5 출시 — 무료·Pro 기본 모델, 8/31까지 도입가 $2/$10

Anthropic 공식 뉴스 · 2026-06-30

Anthropic이 claude-sonnet-5를 공개했다. Sonnet 4.6의 업그레이드로, 내부·서드파티 벤치마크 전반에서 coding, agentic search, multimodal reasoning, professional-task performance의 뚜렷한 향상을 보였다고 밝혔다. 8월 31일까지 도입가 100만 토큰당 입력 $2·출력 $10(이후 $3/$15). Free·Pro의 기본 모델이며 Max·Team·Enterprise, Claude API에서 사용 가능하다. Claude Code에서도 기본 모델로 채택돼 네이티브 1M 토큰 컨텍스트를 지원한다.
