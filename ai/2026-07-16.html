AI 일일 브리핑

2026-07-16 (목) · 최근 24시간 우선, 공식 발표의 진행 중 배포는 최근 7일 예외 포함

TL;DR

  • 지난 24시간의 확인된 신규 항목은 OpenAI의 ChatGPT 맞춤 지침 한도 확대 1건이다.
  • OpenAI의 통합 검색과 Anthropic의 교사용 Claude는 7월 14일 발표로 배포·도입이 진행 중이다.
  • gpt-5.6, Muse Spark 1.1, grok-4.5는 7월 8~9일 공식 출시 후 제공 확대가 이어지는 대형 진행 이슈다.
  • Google과 Mistral은 최근 24시간 안의 검증된 신규 발표가 없었다.

OpenAI

ChatGPT 맞춤 지침 한도, 5,000자로 확대

OpenAI Help Center · 2026-07-15

Plus·Pro·Enterprise·Business·Education 사용자의 맞춤 지침 저장 한도가 1,500자에서 5,000자로 늘었다. 응답 스타일과 행동 규칙을 더 길게 고정할 수 있는 실사용 업데이트다.

“save up to 5,000 characters, up from 1,500”

대화·프로젝트·파일 통합 검색 전면 배포

OpenAI Help Center · 2026-07-14 · 진행 중

웹·iOS·Android의 사이드바에서 대화, 프로젝트, 이미지, 문서를 한 번에 찾고 유형별 필터를 적용할 수 있다. 모든 ChatGPT 요금제에 글로벌 배포됐다는 공식 릴리스 노트를 확인했다.

“Available on all ChatGPT plans globally.”

gpt-5.6 제품군 일반 제공 확대

OpenAI · 2026-07-09 · 대형 진행 이슈

Sol·Terra·Luna 3개 모델이 제한 프리뷰를 거쳐 일반 제공으로 전환됐다. Sol의 ultra는 병렬 에이전트를 조정하며, Responses API에는 프로그램형 도구 호출과 멀티에이전트 베타가 연결된다.

“general availability following our limited preview”

Anthropic

미국 K-12 교사 대상 Claude 무료 프리미엄 제공

Anthropic · 2026-07-14 · 진행 중

검증된 미국 K-12 교사에게 프리미엄 Claude, 교육용 스킬, 50개 주 표준에 연결된 커리큘럼을 무료 제공한다. 수업안·평가 문항·교재 제작을 학교 표준과 연결하는 교육 특화 배포다.

“free access to premium Claude capabilities”

Meta

Muse Spark 1.1과 Meta Model API 공개 프리뷰

Meta AI · 2026-07-09 · 대형 진행 이슈

에이전트 작업, 도구·컴퓨터 사용, 코딩, 멀티모달 이해를 강화한 100만 토큰 컨텍스트 모델이다. Meta AI의 Thinking 모드와 신규 Model API 공개 프리뷰로 개발자 접근이 시작됐다.

“launching a public preview of the new Meta Model API”

xAI

grok-4.5, 코딩·에이전트 작업 중심으로 배포

SpaceXAI · 2026-07-08 · 대형 진행 이슈

Grok Build의 기본 모델이자 Cursor와 xAI API 콘솔에서 제공되는 코딩·지식작업 모델이다. 공식 페이지는 초당 80토큰 제공 속도와 입력 100만 토큰당 2달러, 출력 6달러 가격을 공개했다.

“available today in Grok Build, in Cursor”

Mistral AI · 2026-07-09 · 대형 진행 이슈
