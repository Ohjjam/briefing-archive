AI 일일 브리핑

2026년 6월 8일 (월) · 프런티어 AI 핵심 동향 — 오늘은 Apple WWDC 키노트가 중심, 그 외 주요 진행 스토리(5/28–6/5) 포함

TL;DR

  • Apple WWDC 2026 키노트가 오늘(6/8) 개막 — 전면 개편 Siri가 구글 Gemini 맞춤형 모델로 구동되고, 'Extensions'로 ChatGPT·Gemini·claude를 기본 AI로 선택 가능(보도 기준).
  • Anthropic, Series H로 650억 달러 조달·기업가치 9,650억 달러 — OpenAI 제치고 최고가 스타트업, 비공개 IPO(S-1) 서류 제출(6/1).
  • OpenAI, 생명과학 모델 GPT-Rosalind 신기능 공개(6/3)·'Rosalind Biodefense' 프로그램 확대.
  • xAI, 이미지→영상 grok-imagine-video-1.5-preview API 공개 + 차세대 Grok V9-Medium(1.5조 파라미터) 학습 완료 시사(6/5).
  • Mistral, Le Chat을 'Vibe'로 리브랜딩 + 산업용 물리 AI·파리 인근 데이터센터 발표(AI Now Summit, 5/28).

OpenAI

생명과학 모델 GPT-Rosalind 신기능 공개 + 'Rosalind Biodefense' 확대

OpenAI 공식 블로그 · 2026-06-03 (Biodefense 프로그램 5/29 발표)

OpenAI가 6월 3일 생명과학 전용 모델 GPT-Rosalind의 새 기능을 공개하고, 정부·학계·비영리 바이오보안 개발자에게 모델을 무료 제공하는 'Rosalind Biodefense' 프로그램을 확대했다. "strengthening societal resilience"를 내세워 팬데믹 대비·생물방어 연구를 겨냥한다. MedChemBench·GeneBench에서 gpt-5.5 대비 더 적은 토큰으로 높은 정확도를 보였다고 밝혔다.

오늘 WWDC에서 ChatGPT, Siri 'Extensions' 선택지로 포함

TechCrunch·MacRumors 보도 · 2026-06-08 (WWDC 키노트)

오늘 개막한 애플 WWDC 키노트에서 사용자가 작문·이미지 등 기능의 기본 AI로 ChatGPT를 선택할 수 있는 'Extensions'가 도입된다고 보도됐다. 별도 OpenAI 자체 모델 출시는 지난 24시간 내 없었으며, 가장 최근 제품 발표는 위 GPT-Rosalind(6/3)다. (상세는 아래 '기타' 카드 참조)

Google (Gemini · DeepMind)

구글 Gemini, 애플 새 Siri의 두뇌로 채택 (연 ~$10억 규모 보도)

TechCrunch·MacRumors 보도 · 2026-06-08 (WWDC)

오늘 개막한 WWDC에서 전면 개편된 Siri가 구글의 맞춤형 Gemini 모델(보도상 약 1.2조 파라미터)로 구동된다고 알려졌다. 연 약 10억 달러 규모 라이선스로 보도됐다. 구글이 프런티어 경쟁사 플랫폼에 핵심 추론을 공급하는 대형 사례로, OS 단 AI 유통 경쟁의 분수령이다.

gemini-3.5 Pro 일반 공개 임박

Google I/O 2026 · 원 발표 2026-05-19, 6월 공개 예정

5월 19일 I/O에서 공개된 gemini-3.5 제품군 중 Flash는 출시됐고, Pro는 내부 테스트를 거쳐 6월 일반 공개가 예상된다. Ultra 구독료는 월 $250→$200로 인하됐고 개발자용 $100 티어가 신설됐다. (원 발표 2026-05, 6월 재조명)

Anthropic

Series H로 650억 달러 조달·기업가치 9,650억 달러 — OpenAI 추월

Anthropic 공식 뉴스룸 · 2026-05-28

Anthropic이 시리즈 H에서 650억 달러를 조달하며 포스트머니 기업가치 9,650억 달러를 기록, OpenAI를 제치고 세계 최고가 스타트업이 됐다. 2월 시리즈 G(3,800억 달러)의 약 2.5배다. 런레이트 매출은 470억 달러를 넘어섰다고 밝혔다. (주요 진행 스토리)

SEC에 비공개 IPO 서류(S-1) 제출

Anthropic 공식 뉴스룸 · 2026-06-01

6월 1일 Anthropic이 SEC에 Form S-1 초안을 비공개 제출했다고 공식 발표했다. 심사 후 이르면 올가을 상장 가능성이 거론되며, OpenAI보다 먼저 상장에 이를 수 있다는 전망이 나온다. 오늘 WWDC에서는 claude가 Siri 'Extensions' 선택지 중 하나로 포함됐다.

Meta (Llama · Meta AI)

xAI (Grok)

이미지→영상 모델 grok-imagine-video-1.5-preview API 공개

xAI 공식(x.ai) · 2026-06

xAI가 정지 이미지를 자연어 모션 제어로 시네마틱 영상으로 변환하는 grok-imagine-video-1.5-preview를 API 프리뷰로 공개했다. 최대 720p, 원본의 조명·룩을 유지한다. 함께 Grok Web에 SharePoint·Outlook·Notion·GitHub·Linear 등 Connectors와 사용자 MCP 서버 연결을 추가했다.

차세대 Grok V9-Medium(1.5조 파라미터) 학습 완료 시사[미확인]

Elon Musk X 게시물 보도 · 2026-06-05

머스크가 6월 5일 차세대 Grok V9-Medium(1.5조 파라미터, 현 v8-small의 3배) 학습 완료를 시사했다. 지도 미세조정(SFT)·강화학습(RL)이 진행 중이며 추론·코드 성능 향상이 기대된다. 공식 릴리스 전 단계로 세부 사양은 미확정이다.

Mistral AI

Le Chat을 'Vibe'로 리브랜딩 + 산업용 물리 AI·데이터센터 발표

Mistral 공식(mistral.ai) · 2026-05-28 (AI Now Summit, 파리)

Mistral이 AI Now Summit(5/28)에서 어시스턴트 Le Chat을 'Vibe'로 통합 리브랜딩하고 Work Mode·Code Mode를 갖춘 단일 에이전트로 확장했다. 항공·자동차·반도체를 겨냥한 산업 AI 스택(Airbus·BMW·ASML 고객)과 파리 인근 10MW 추론 데이터센터(Q3 가동)를 공개했다. voxtral-tts 음성 모델과 오픈소스 Search Toolkit도 함께 발표. (주요 진행 스토리)
