AI 일일 브리핑

2026년 6월 19일 (금) · 최근 24~48시간 프론티어 AI 동향 · OpenAI · Google · Anthropic · Meta · xAI · Mistral

TL;DR

  • OpenAI가 ChatGPT에 예약 작업(Scheduled Tasks)을 정식 출시 — 유료 플랜에서 알림·반복 작업·웹 모니터링을 자동화 (6/17).
  • Databricks Data + AI Summit(6/15~18)에서 xAI Grok과 Anthropic Claude Fable 5가 Agent Bricks에 네이티브로 탑재됨 (6/18).
  • Google가 Gemini CLI를 Antigravity CLI로 전환 시작 — 단일 멀티에이전트 플랫폼으로 통합 (6/18).
  • Meta는 지난 24~48시간 내 주요 발표 없음. Mistral도 신규 모델 발표는 없으나 약 35억 달러 규모 투자 유치 협상이 진행 중.

OpenAI

ChatGPT '예약 작업(Scheduled Tasks)' 정식 출시 — 반복 작업·웹 모니터링 자동화

OpenAI Help Center · 2026-06-17

유료(Plus·Pro·Business·Enterprise) 사용자가 ChatGPT에 알림, 반복 업무, 웹·연결 앱 변화 모니터링을 맡길 수 있게 됐다. 사이드바의 새 Scheduled 페이지에서 활성 작업과 다음 실행 시점을 한눈에 보고 일시정지·수정·삭제할 수 있으며, 모니터링 작업은 "notify users only when there is something worth reporting" 방식으로 의미 있는 변화가 있을 때만 알린다. 플랜 등급별로 활성 작업 수 제한이 다르다. gpt-5.5

최근 누적 변경: GPT-5.5 Instant 개인화 확대 · 모델 선택기 개편 · GPT-5.2 은퇴

OpenAI Help Center · ChatGPT Release Notes (2026-06-09~12, 재조명)

48시간 밖 변경이지만 맥락상 정리: 기본 모델 gpt-5.5 Instant의 개인화·정확도 개선이 Go·Free까지 확대(6/9)됐고, 모델 선택기가 모델 카탈로그 대신 '속도-깊이 슬라이더'로 바뀌었다(6/10). gpt-5.2 계열(Instant·Thinking·Pro)은 6/12자로 ChatGPT에서 은퇴, 기존 대화는 대응되는 gpt-5.5 모델로 자동 이전된다.

Google

Gemini CLI·Code Assist를 Antigravity CLI로 전환 시작

Google Developers Blog · 2026-06-18

개인·AI Pro·AI Ultra 등급의 Gemini Code Assist IDE 확장과 Gemini CLI가 6/18부터 요청 처리를 중단하고, Google이 도구를 단일 멀티에이전트 플랫폼 Antigravity로 통합한다. Antigravity CLI가 이미 제공되며, 기존 Gemini CLI 사용자는 새 CLI로 이전하게 된다. 에이전트형 개발 환경으로의 일원화가 핵심. antigravity-cli

Workspace, Gemini 앱 관리자 제어(admin controls) 롤아웃 완료

Google Workspace Updates · 2026-06-17~18

Rapid Release·Scheduled Release 도메인 대상 전면 롤아웃이 6/17 시작돼 6/18 무렵 완료 예정. 관리자가 조직 내 Gemini 앱 사용을 제어할 수 있는 설정이 배포되며, 엔터프라이즈 거버넌스 측면에서 Gemini가 Workspace에 더 깊게 결합되는 흐름. gemini-3.5-flash

Anthropic

Claude Fable 5, Databricks Agent Bricks에 Unity AI Gateway로 탑재

Databricks Blog (Agent Bricks, DAIS 2026) · 2026-06-18

Databricks가 Anthropic 최신 모델 claude-fable-5를 Unity AI Gateway를 통해 플랫폼에 들였다. 거버넌스·가시성·비용 통제를 유지한 채 엔터프라이즈 워크플로에서 Claude를 쓸 수 있게 하는 통합으로, Agent Bricks는 OpenAI·Gemini·Grok·Kimi와 함께 Claude까지 모델 선택지를 넓혔다.

Claude Code 안정화 패치 · Claude Design 업데이트[미확인]

Anthropic Newsroom (릴리스 노트 집계) · 2026-06-18

릴리스 노트 집계 기준 Claude Code가 스트림 중단·멈춤 스피너·WSL2 스크롤·서브에이전트 트랜스크립트·원격 작업 상태·플러그인 로딩 성능 이슈를 수정했고, Claude Design은 디자인 시스템 정합성과 Claude Code 동기화, 캔버스 직접 편집을 강화했다고 전해진다. 1차 릴리스 노트 URL을 특정하지 못해 [미확인]으로 표기.

xAI

Grok, Databricks Agent Bricks에 네이티브 탑재 (SpaceX 파트너십)

Databricks Blog (Agent Bricks, DAIS 2026) · 2026-06-18

6/18 Databricks Data + AI Summit에서 grok 모델이 Agent Bricks에 네이티브로 제공된다고 발표됐다. Databricks가 SpaceX와 파트너십을 맺어 Grok(및 Kimi)을 OpenAI·Anthropic·Gemini와 나란히 모델 선택지에 추가하는 형태로, 엔터프라이즈 에이전트 빌드에서 Grok 접근성이 크게 넓어졌다.

xAI Speech-to-Text API 정식 출시 · Grok Voice 공개

xAI Docs · Release Notes (2026-06-04~15, 재조명)

48시간 밖이지만 맥락 정리: Grok Voice가 6/4 공개돼 음성 대화·음성 응답이 가능해졌고, xAI Speech-to-Text API가 6/15자로 정식 출시되어 25개 언어 배치·스트리밍 전사를 지원한다. Grok이 음성·영상·도구 연결로 업무용 플랫폼화하는 흐름.

Databricks Data + AI Summit 2026 — 멀티모델 에이전트 경쟁의 무대

Databricks Blog · 2026-06-15~18

3만 명 이상이 모인 이번 서밋에서 Agent Bricks가 모델 선택지(OpenAI·Anthropic·Gemini·Grok·Kimi)·임의 하니스(LangGraph·CrewAI·Claude Code SDK)·Lakebase 기반 에이전트 메모리로 확장됐다. 프론티어 랩들이 자사 모델을 엔터프라이즈 에이전트 플랫폼에 꽂아 넣는 경쟁 구도가 선명해졌다. agent-bricks
