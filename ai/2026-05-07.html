OpenAI

ChatGPT 기본 모델 GPT-5.5 Instant 전면 적용

TechCrunch · 2026-05-05

OpenAI가 Free/Plus/Pro 전체 사용자 대상으로 ChatGPT 기본 모델을 GPT-5.5 Instant로 교체했다. 의료·법률·금융 등 고난도 프롬프트에서 환각 클레임이 이전 GPT-5.3 Instant 대비 52.5%, 사용자가 사실 오류로 신고한 대화에서는 37.3% 감소. 기존 유료 사용자는 3개월 동안 5.3 Instant 액세스 유지.

개인화·메모리 출처 표시 기능 도입

Axios · 2026-05-05

GPT-5.5 Instant는 검색 도구로 과거 대화·파일·Gmail을 자동 참조해 개인화된 답변을 제공한다(웹 Plus/Pro 우선 출시). 모든 모델에서 메모리 출처가 표시되며, 사용자가 잘못된 출처를 직접 삭제·수정 가능. 불필요한 이모지 사용도 줄였다.

Google / DeepMind

Deep Research Max 공개 — Gemini 3.1 Pro 기반 자율 리서치 에이전트

Google Blog · 2026-04월말 (현재 진행)

Google DeepMind가 Gemini 3.1 Pro 기반의 Deep Research / Deep Research Max 두 가지 에이전트를 출시. Max는 MCP 지원·네이티브 시각화·확장된 test-time compute를 통한 반복적 추론·검색·리포트 정제로 비동기 백그라운드 워크플로 최적화. 일반 Deep Research는 인터랙티브 사용성·낮은 지연 비용에 특화.

"a step change for autonomous research agents ... unprecedented analytical quality to long-horizon research workflows"

Gemini Embedding 2 GA · Workspace Intelligence 컨텍스트 통합

Gemini API Changelog · 2026-05월 초

Gemini Embedding 2가 Gemini API와 Vertex AI에서 정식 출시 — 텍스트·이미지·비디오·오디오 멀티모달 임베딩이 안정 버전으로 제공된다. 동시에 Workspace Intelligence는 Gmail/Chat/Calendar/Drive의 실시간 컨텍스트를 생성 작업에 자동 주입하며, 관리자는 Admin 콘솔에서 데이터 소스를 제어할 수 있다.

Anthropic

Claude Opus 4.7 일반 출시 — 코딩·비전 강화, 가격 동결

Anthropic News · 2026-05월 초

claude-opus-4-7이 GA. 소프트웨어 엔지니어링과 장기 실행 코딩 태스크 성능, 고해상도 이미지 비전이 향상되었다. 가격은 Opus 4.6과 동일한 입력 $5 / 출력 $25 per 1M tokens. 동시에 Claude Design(시각 산출물 협업 도구)을 Anthropic Labs로 공개.

금융 서비스 10종 에이전트 템플릿 + Microsoft 365 풀 통합

Fortune · 2026-05-05

Anthropic이 피치북 작성·KYC 스크리닝·월말 결산 등 금융권 시간 소모 업무를 자동화하는 10개 에이전트 템플릿을 공개. Excel·PowerPoint·Word·Outlook을 가로지르는 Microsoft 365 add-in 출시(Outlook은 곧 추가)로 앱 간 컨텍스트가 자동 이어진다. Moody's 데이터 파트너십도 발표.

Claude Security 공개 베타 · Managed Agents Memory · API 레이트리밋 2배

Releasebot · 2026-05-06

Claude Enterprise용 Claude Security 공개 베타: Opus 4.7 기반 코드 취약점 스캔·자동 수정 제안, 스케줄·타깃 스캔, 트리아지·익스포트 개선. Managed Agents에 학습형 메모리 레이어 추가. 5월 6일부로 Claude Code/API 레이트리밋이 상향되어 Pro/Max/Enterprise의 5시간 한도가 두 배로. SpaceX Colossus 데이터센터 활용 파트너십도 발표.

Meta

LlamaCon 발표 라인업 후속 — Llama 4 Scout/Maverick 보급 확대

Meta AI Blog · 2026-04-29 (LlamaCon)

Meta는 4월 말 LlamaCon에서 공개한 Llama 4 Scout(MoE, 문서 요약·추론 특화)과 Llama 4 Maverick(범용 멀티모달 챗) 보급을 5월에도 이어가는 중. Hugging Face에서 가중치 다운로드 가능하며, WhatsApp/Messenger/Instagram Direct/Meta.AI에서 즉시 체험 가능. 지난 24시간 내 신규 모델 발표는 확인되지 않았다.

xAI

grok-voice-think-fast-1.0 Voice Agent API + Speech APIs 공개

xAI Docs · 2026-05월 초

xAI가 복잡한 다단계 워크플로·고객지원·세일즈·엔터프라이즈 용도의 Voice Agent API 모델 grok-voice-think-fast-1.0을 공개. 25개 이상 언어 지원, 저지연·정확한 도구 호출이 강점. 같은 시기 Grok Speech-to-Text·Text-to-Speech 독립 API도 출시 — 실시간/배치 엔드포인트, 화자 분리, 타임스탬프, 표현형 음성 태그 포함.

Business Standard · 2026-05-04
